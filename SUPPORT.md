# Support

Thanks for using **GLM for GitHub Copilot Chat**. Here is how to get help.

## Before you ask

- Read the [README](README.md) — it covers installation, settings, the model table, and the
  difference between the Coding Plan and Standard APIs.
- Make sure you are on the latest version from the VS Code Marketplace.
- Run **GLM: Show Logs** from the Command Palette (enable `glm-copilot.debugLogging` first) and
  check the output channel for errors.

## Questions and discussion

For "how do I…" questions and general discussion, open a
[GitHub Discussion](https://github.com/KiwiGaze/glm-for-copilot/discussions).

## Bugs and feature requests

- **Found a bug?** Open a [Bug report](https://github.com/KiwiGaze/glm-for-copilot/issues/new?template=bug_report.yml).
- **Want a feature?** Open a [Feature request](https://github.com/KiwiGaze/glm-for-copilot/issues/new?template=feature_request.yml).

## GLM API and account issues

Problems with your GLM API key, billing, your Coding Plan subscription, or model availability
are handled by Z.ai / Zhipu, not by this extension. See the official
[GLM API documentation](https://docs.z.ai) and your provider dashboard:

- International: [z.ai](https://z.ai/manage-apikey/apikey-list)
- Mainland China: [open.bigmodel.cn](https://open.bigmodel.cn/usercenter/proj-mgmt/apikeys)
